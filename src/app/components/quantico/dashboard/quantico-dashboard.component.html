Reglas y Convenciones

Estructura de archivos

src/app/components/
  quantico/                    # Componentes exclusivos de Quantico
    dashboard/
      quantico-dashboard.component.ts
      quantico-dashboard.component.html
      quantico-dashboard.component.scss

src/styles/
  _tokens.scss                 # Design tokens (prefix $qt-)

Reglas de CSS

  • BEM naming: .qt-componente__elemento--modificador
  • NUNCA hardcodear colores hex — siempre usar variables $qt-* de _tokens.scss
  • Import de tokens: @import '../../../../styles/tokens';
  • Componentes standalone: siempre standalone: true, sin NgModules
  • Selector prefix: siempre app-

Independencia de entornos

  • Quantico es independiente — no comparte componentes con SocialGest, AdvocatesPro ni Tikket
  • Todos los tokens usan prefijo $qt-
  • Los componentes van en src/app/components/quantico/

Accesibilidad

  • Todos los elementos interactivos con aria-label
  • Soporte de teclado: Tab, Enter, Space, Escape, Arrow keys
  • Estados :focus-visible para usuarios de teclado
  • Contraste WCAG AA minimo
