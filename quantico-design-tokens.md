  }

  }

    background: $qt-button-primary-disabled-bg;
  }
}

.qt-card {

  border-radius: $qt-card-border-radius;
  padding: $qt-card-padding-simple;
  color: $qt-card-title-color;
}

.qt-table {
  background: $qt-table-bg;

  }

  }

## Naming Convention

All Quantico tokens follow the pattern:
```
$qt-{component}-{property}
```

- **Color scales**: `$qt-{category}-{shade}` — e.g. `$qt-primary-500`, `$qt-grey-800`
- **Component tokens**: `$qt-{component}-{property}` — e.g. `$qt-button-primary-bg`, `$qt-card-border-radius`
- **shade**: `50` (lightest) to `900` (darkest), or `base` for single-value scales
